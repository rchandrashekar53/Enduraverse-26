# Enduraverse-26 Hackathon 🚀

Welcome to **Enduraverse-26**, the official repository for the Hackathon organized by **Endurance Technologies Limited**.  
This event is designed to foster innovation, collaboration, and creativity in the fields of automotive technology, electronics, and software systems.

---

## 🎯 Objective
The hackathon aims to:
- Encourage **indigenous innovation** in automotive and brake control systems.
- Build **scalable solutions** that benchmark against global standards.
- Promote **collaboration** between industry professionals, students, and researchers.
- Explore **cutting-edge technologies** in ABS, traction systems, and vehicle dynamics.

## 🏆 Judging Criteria
Projects will be evaluated based on:
- **Innovation**: Novelty and creativity of the solution.  
- **Technical Depth**: Robustness, scalability, and engineering quality.  
- **Impact**: Practical applicability and potential industry adoption.  
- **Presentation**: Clarity, engagement, and storytelling.
