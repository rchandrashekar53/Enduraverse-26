# 🔥 DreamVision – Smart Industrial Monitoring System

A real-time AI-powered industrial monitoring system that integrates IoT devices, cloud backend, and machine learning to detect anomalies and predict machine behavior for smart factories.

---

## 🚀 Features
- 📷 Live ESP32 Camera Feed
- 🌡️ Real-time Temperature Monitoring
- ☁️ Supabase Cloud Data Storage
- 🧠 AI-Based Prediction & Anomaly Detection
- 📊 Interactive Streamlit Dashboard
- 🚨 OK/NOK Machine Status Classification

---

```bash
# Clone the repository
